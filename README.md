## Getting Started

### Prerequisites

-   Node.js 18+
-   npm or bun

### Installation

```bash
# Clone the repository
git clone https://github.com/centralcoordinatordesk-rgb/cisa-career-boost.git

# Navigate to project directory
cd cisa-career-boost

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```
